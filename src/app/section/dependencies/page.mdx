# Quickly Install Dependencies

<div className="space-y-2">

TL;DR? Here's the all-in-one:

    ```bash
    pnpm add -D prettier prettier-plugin-tailwindcss tailwindcss-animate eslint-config-prettier @trivago/prettier-plugin-sort-imports eslint-plugin-check-file eslint-plugin-n
    ```

</div>

<div className="space-y-4">

Not sure you need everything? Take them one by one:

- Prettier ([prettier](https://github.com/prettier/prettier))

  ```bash
  pnpm add -D prettier
  ```

- Tailwind Prettier Plugin ([prettier-plugin-tailwindcss](https://github.com/tailwindlabs/prettier-plugin-tailwindcss))

  ```bash
  pnpm add -D prettier-plugin-tailwindcss
  ```

- Tailwind Animate Plugin ([tailwindcss-animate](https://github.com/jamiebuilds/tailwindcss-animate))

  ```bash
  pnpm add -D tailwindcss-animate
  ```

- ESLint configuration with Prettier ([eslint-config-prettier](https://github.com/prettier/eslint-config-prettier))

  ```bash
  pnpm add -D eslint-config-prettier
  ```

- Import Order Sort Plugin ([@trivago/prettier-plugin-sort-imports](https://github.com/trivago/prettier-plugin-sort-imports))

  ```bash
  pnpm add -D @trivago/prettier-plugin-sort-imports
  ```

- File Checking Plugin ([eslint-plugin-check-file](https://github.com/dukeluo/eslint-plugin-check-file))

  ```bash
  pnpm add -D eslint-plugin-check-file
  ```

- Prevent using 'process.env' directly ([eslint-plugin-n](https://github.com/eslint-community/eslint-plugin-n))

  ```bash
  pnpm add -D eslint-plugin-n
  ```

</div>
